# SpringPhotoGallery

クラウド対応画像ストアアプリ"[OsaStagram2][1]"のサーバーアプリ"[PhotoGallery][2]"を、SpringFrameworkを使用することによってリファクタリングしました。
今回のサーバーアプリ対応では、クライアントに現在開発中の"[OsaStagram2]"を検討してください。
